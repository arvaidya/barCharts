# barCharts
bar charts example using Reactive js

Installation: 

1. open npm console
2. clone the git url 
3. navigate to project folder
4. run npm install

Running project:

1. run command 'grunt serve' in npm console
2. in browser open 'http://localhost:9999/views/index.html'

Testing project:

1. in npm console run command 'npm test'
